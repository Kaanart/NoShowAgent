# Technology Stack

## Core Languages
- **Python:** Used for the machine learning prediction model and core backend logic.
- **TypeScript:** Powers the robust, type-safe frontend dashboards.

## Backend Frameworks
- **FastAPI / Django:** Python-based frameworks for the ML model serving and backend administration.
- **Express.js (Node.js):** Potential lightweight middleware or BFF (Backend-For-Frontend) layer.

## Frontend Framework
- **React:** The industry standard for dynamic, data-heavy dashboards, providing schedulers and technicians with an interactive UI.

## Database & Data Warehousing
- **Google Cloud BigQuery:** Serves as the primary data warehouse and analytics database, essential for processing large volumes of hospital data and training the ML-driven no-show prediction model.
